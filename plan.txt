我们正在做“隔空弹奏乐器”项目。目标是用摄像头识别手势切换乐器，并根据手势演奏，由扬声器播放声音，屏幕/OLED/LED 显示状态。

当前最终策略：
电脑主控 + Python 手势识别桥 + 浏览器前端 + ESP32/Arduino 辅助硬件 + 外接摄像头 + 小显示屏/OLED/LED + 扬声器。

电脑负责：
- Python 识别桥独占摄像头并输出 WebSocket 手势/和弦/乐器消息
- 乐器切换逻辑
- 浏览器端 Web Audio 播放和弦
- 浏览器主 UI 显示骨架、根音区、乐器区、和弦轮盘

ESP32/Arduino 负责：
- LED 灯效反馈
- 物理按钮，比如开始、重置、切换模式
- 旋钮调节音量
- OLED 显示当前乐器
- 可选震动马达或灯带反馈

推荐四人分工：
A：摄像头与手势识别，负责 OpenCV + MediaPipe Hands，输出手势数据。
B：乐器切换与演奏逻辑，负责状态机、防误触、乐器切换和演奏事件。
C：音频播放与主界面，负责 Web Audio、HTML/CSS/JS UI、WebSocket 消息消费。
D：ESP32/Arduino 硬件交互，负责按钮、旋钮、OLED、LED、串口通信和硬件接线。

推荐软件技术栈：
Python 3.12（conda env: mediapipe-hands-instrument）
OpenCV
MediaPipe Hands
NumPy
websockets
Web Audio API
HTML/CSS/JS
YAML/JSON
Git

推荐串口通信：
PC -> ESP32:
INST:piano;CHORD:C major;STATE:PLAYING

ESP32 -> PC:
BTN:start;VOL:72
