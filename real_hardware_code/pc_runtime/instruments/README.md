# Instruments

Put reusable instrument sound libraries here so MaixCAM2 demos can use stable paths.

## Layout

```text
instruments/
  drums/
    Shino Drums(sf2)/
      02_Shino_Snare.sf2
      snare_samples/
        ghost.wav
        normal.wav
        accent.wav
  pianos/
    SalamanderGrandPiano-V3/
      SalamanderGrandPiano-V3+20200602.sf2
      samples/
        036.wav
        ...
        095.wav
  guitars/
    Bread Breads Distortion Guitar/
      Bread Breads Distortion Guitar v2.1.sf2
      strums/
        C5_down.wav
        C5_up.wav
    acoustic_guitar/
      FSS-SteelStringGuitar-20200521.sf2
      strums/
        C_down.wav
        C_up.wav
        G_down.wav
        G_up.wav
  samples/
    drums/
      kick.wav
      snare.wav
```

## Notes

- `pc_bread_drum_server.py` defaults to `instruments/drums/bread_breads_drum_kit/Bread Breads Drum Kit1.03.1.sf2`.
- `pc_synth_server.py` defaults to `instruments/pianos/fluid_r3/FluidR3_GM.sf2`.
- `pc_instrument_server.py` uses `instruments/drums/Shino Drums(sf2)/02_Shino_Snare.sf2` plus pre-rendered `snare_samples/ghost.wav`, `normal.wav`, and `accent.wav` for snare drum mode, `instruments/pianos/SalamanderGrandPiano-V3/SalamanderGrandPiano-V3+20200602.sf2` plus pre-rendered `samples/036.wav` through `samples/095.wav` for piano, `instruments/guitars/Bread Breads Distortion Guitar/Bread Breads Distortion Guitar v2.1.sf2` for electric guitar, and `instruments/guitars/acoustic_guitar/FSS-SteelStringGuitar-20200521.sf2` for acoustic guitar.
- Regenerate Salamander piano samples with `python pc_generate_piano_wavs.py` after changing the piano SoundFont.
- Regenerate Shino snare samples with `python pc_generate_snare_wavs.py` after changing the snare SoundFont.
- Electric and acoustic guitar modes play pre-rendered strum WAV files first. Regenerate all guitar strums with `python pc_generate_guitar_strum_wavs.py` after changing either guitar SoundFont.
- Large `.sf2` and `.wav` files are intentionally not generated by the code. Copy or download the files into these folders.
